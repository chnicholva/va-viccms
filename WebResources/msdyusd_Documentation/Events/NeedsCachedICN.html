Evoked through JavaScript window.open calls when a record that should have a value in its Cached ICN lookup field but does not. Action calls beneath a control's respective events should create a new ftp_cachedicn record and link it to the event's parent record.

The following action call records should be added underneath this event, to handle the aforementioned logic:
"Record Needs Cached ICN: Create Cached ICN Record"
"Record Needs Cached ICN: Link new Cached ICN to Record"

Expects the following parameters:

ICN A veteran ICN, used to create a new ftp_cachedicn record
logicalName Entity name of the record calling this event, to which a new Cached ICN record will be linked
recordId GUID of the records calling this event, to which a new Cached ICN record will be linked
